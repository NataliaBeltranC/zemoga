# Steps

1. `npm i`
2. `npm start`
3. `npm run test`

# Consider

1. If you get an error like "Node Sass could not find a binding for your current environment" run `npm rebuild node-sass`
2. Remember the version of the `.nvmrc` and then `nvm use vx.x.x`
